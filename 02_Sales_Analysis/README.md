# Sales Analysis

## Project Overview

This project analyzes sales data using Python to understand business performance and identify important trends in sales.

The analysis helps answer questions like:

* Which products sell the most?
* What is the total sales revenue?
* Which categories perform better?

## Tools and Libraries Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

## Dataset

The dataset contains sales transaction details.

Example columns:

* Product
* Category
* Quantity
* Price
* Total Sales
* Date

## Analysis Performed

The following analysis was done in this project:

* Total sales calculation
* Product-wise sales analysis
* Category-wise performance
* Sales distribution
* Identifying top-selling products

## Visualizations

The project includes several visualizations such as:

* Bar chart for product sales
* Pie chart for category distribution
* Line chart for sales trends
* Histogram for sales distribution
