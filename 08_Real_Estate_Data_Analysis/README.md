Real Estate Data Analysis
📌 Project Overview

This project analyzes real estate housing data using Python to understand property prices, house sizes, and bedroom distribution across different cities.
The analysis helps identify cities with the highest property prices, largest house areas, and overall housing trends.

The project also visualizes housing data using multiple charts to make insights easier to understand.

📊 Technologies Used

Python

NumPy

Pandas

Matplotlib

Seaborn

📈 Visualizations

The project includes several visualizations such as:

Bar Plot – House prices by city

Bar Plot – House area (square feet) comparison

Bar Plot – Number of bedrooms by city

Pie Chart – Price percentage distribution by city

📂 Dataset

The dataset contains the following columns:

HouseID

City

Area_sqft

Bedrooms

Price

⚙️ Features

Total number of houses available for sale

Cities where houses are available

House with the largest area

House with the smallest area

House with the highest price

House with the lowest price

Average house price

Data visualization dashboard
