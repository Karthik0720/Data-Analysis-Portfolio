HR Employee Data Analysis
📌 Project Overview

This project analyzes HR employee data using Python to understand employee distribution across departments, salary trends, experience levels, and attrition status.

The analysis helps identify which department has the highest salary, employee experience patterns, and the number of employees leaving or staying in the organization.

📊 Technologies Used

Python

NumPy

Pandas

Matplotlib

Seaborn

📈 Visualizations

The project includes multiple visualizations such as:

Bar Plot – Average salary by department

Bar Plot – Average experience by department

Count Plot – Employee attrition (Yes/No)

Pie Chart – Department-wise salary distribution

📂 Dataset

The dataset contains the following columns:

EmployeeID

Department

Salary

Experience

Attrition

⚙️ Features

Total number of departments

Average salary by department

Highest salary department

Lowest salary department

Average employee salary

Employee experience details

Employee with highest experience

Employee with lowest experience

Attrition analysis (working vs not working employees)

Data visualization dashboard
