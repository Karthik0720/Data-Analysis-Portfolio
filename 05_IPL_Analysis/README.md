IPL Data Analysis

📌 Project Overview

This project analyzes cricket match data from the Indian Premier League (IPL) using Python. The goal of this analysis is to understand team performance by examining matches played, runs scored, wickets taken, and match results. The project performs data exploration and visualization to reveal insights such as which team played the most matches, which team scored the highest runs, and the comparison between winning and losing teams.

📊 Technologies Used

Python
NumPy
Pandas
Matplotlib
Seaborn

📈 Visualizations

The project includes multiple visualizations such as:

Bar Plot – Number of matches played by each team
Bar Plot – Total runs scored by each team
Bar Plot – Total wickets taken by each team
Bar Plot – Team win vs lose comparison

📂 Dataset

The dataset contains the following columns:

MatchID
Team
Opponent
Runs
Wickets
Result

⚙️ Features

Count total number of teams in the dataset
Display names of all teams
Calculate number of matches played by each team
Identify the team with the highest total runs
Identify the team with the lowest total runs
Determine the team with the most wickets taken
Determine the team with the least wickets taken
Find teams that won matches
Find teams that lost matches
Visualize team performance using charts
