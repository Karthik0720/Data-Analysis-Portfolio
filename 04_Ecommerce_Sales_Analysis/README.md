E-Commerce Data Analysis
📌 Project Overview

This project analyzes E-commerce sales data using Python to understand product performance, customer purchasing patterns, and sales distribution.
The analysis includes data exploration and visualization to identify key insights such as the most expensive product, most ordered customer, and total sales spend.

📊 Technologies Used

Python

NumPy

Pandas

Matplotlib

Seaborn

📈 Visualizations

The project includes multiple visualizations such as:

Line Plot – Product quantity purchased

Bar Plot – Product price comparison

Bar Plot – Total product spending

Pie Chart – Product sales distribution

📂 Dataset

The dataset contains the following columns:

CustomerName

Product

Category

Quantity

Price

Additional calculated column:

TotalSpend = Quantity × Price

⚙️ Features

Identify the most expensive product

Find the most expensive category

Determine the most ordered customer

Identify the highest selling product

Calculate average product price

Visualize sales data using charts

🚀 How to Run the Project

1️⃣ Clone the repository

2️⃣ Install required libraries

3️⃣ Run the Python file

Example:

python ecommerce_analysis.py
