# Student Performance Analysis

## Project Overview

This project analyzes student performance data using Python.
The goal is to understand how students perform in different subjects and identify patterns in their scores.

## Tools and Libraries Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

## Dataset

The dataset contains information about student scores in different subjects.

Example columns:

* Student Name
* Maths Score
* Science Score
* English Score
* Total Score

## Analysis Performed

The following analysis was done in this project:

* Calculate average score of students
* Find highest and lowest scores
* Subject-wise performance analysis
* Distribution of marks
* Comparison between subjects

## Visualizations

The project includes the following visualizations:

* Histogram of student scores
* Boxplot for subject distribution
* Bar chart for average scores
* Scatter plot for subject comparison
