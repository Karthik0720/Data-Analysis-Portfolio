# COVID-19 Data Analysis

## 📌 Project Overview

This project analyzes COVID-19 data using Python to understand the spread of the virus across different countries.
It performs data exploration and visualization to study confirmed cases, recoveries, and deaths.

## 📊 Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

## 📈 Visualizations

The project includes multiple visualizations such as:

* Line Plot – Confirmed cases by country
* Bar Plot – Death cases by country
* Bar Plot – Recovered cases by country
* Scatter Plot – Comparison of Confirmed, Recovered, and Deaths
* Histogram – Distribution of confirmed cases
* Pie Chart – Country-wise confirmed cases

## 📂 Dataset

The dataset contains the following columns:

* Date
* Country
* Confirmed
* Recovered
* Deaths

## ⚙️ Features

* Country-wise COVID statistics
* Most affected country
* Least affected country
* Recovery statistics
* Death statistics
* Data visualization dashboard

## 🚀 How to Run the Project

1. Clone the repository
2. Install required libraries
3. Run the Python file

Example:

```bash
python covid_analysis.py
```
