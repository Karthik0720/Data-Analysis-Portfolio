Netflix Data Analysis

📌 Project Overview

This project analyzes Netflix movie data using Python to understand movie popularity, ratings, and audience engagement. The analysis focuses on identifying highly rated movies, movies with the most reviews, and the genres that people like the most.

The project performs data exploration and visualization to gain insights about movie performance on the streaming platform Netflix.

📊 Technologies Used

Python
NumPy
Pandas
Matplotlib
Seaborn

📈 Visualizations

The project includes multiple visualizations such as:

Line Plot – Movie reviews trend
Bar Plot – Movie rating comparison
Horizontal Bar Plot – Average rating by genre
Pie Chart – Movie rating distribution

📂 Dataset

The dataset contains the following columns:

Movie
Genre
Rating
Reviews

⚙️ Features

Identify the highest rated movie
Identify the lowest rated movie
Find the movie with the highest number of reviews
Find the movie with the lowest number of reviews
Determine the most liked movie genre based on average rating
Determine the least liked movie genre
Calculate the average movie rating
Visualize movie performance using charts

🚀 How to Run the Project

1️⃣ Clone the repository

2️⃣ Install required libraries

3️⃣ Run the Python file

Example:

python netflix_analysis.py
