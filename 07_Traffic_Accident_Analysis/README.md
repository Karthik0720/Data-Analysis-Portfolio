Traffic Accident Data Analysis
📌 Project Overview

This project analyzes traffic accident data using Python to understand accident patterns across different cities.
The analysis focuses on identifying cities with the highest injuries and deaths, weather conditions during accidents, and vehicle involvement.

The project also visualizes accident statistics using different charts to make the insights easy to understand.

📊 Technologies Used

Python

NumPy

Pandas

Matplotlib

Seaborn

📈 Visualizations

The project includes multiple visualizations such as:

Bar Plot – Accident injuries by city

Bar Plot – Accident deaths by city

Line Plot – Comparison of injuries and deaths

Pie Chart – Vehicles involved in accidents

📂 Dataset

The dataset contains the following columns:

City

Weather

Injuries

Deaths

VehiclesInvolved

⚙️ Features

Total number of accidents recorded

Cities where accidents occurred

City with the highest injuries

City with the lowest injuries

City with the highest deaths

City with the lowest deaths

Weather conditions during accidents

Average number of deaths

Data visualization dashboard
