Stock Market Data Analysis
📌 Project Overview

This project analyzes stock market data using Python to understand price trends, trading volume, volatility, and daily returns across multiple companies.

The analysis helps identify which stocks perform better, which are more volatile, and how trading activity differs between companies. The project also presents the results using a black-and-white professional dashboard.

📊 Technologies Used
Python
NumPy
Pandas
Matplotlib
Seaborn

📈 Visualizations

The project includes multiple financial visualizations such as:

Line Plot – Average stock high price by company
Line Plot – Total trading volume comparison
Bar Plot – Stock volatility analysis
Bar Plot – Average daily return comparison
Bar Plot – Total trading value by stock
Pie Chart – Return percentage distribution

📂 Dataset

The dataset contains the following columns:

Date
Stock
Open
High
Low
Close
Volume
Additional calculated columns:
TradingValue = Close × Volume
Volatility = High − Low
DailyReturn = Close − Open

⚙️ Features
Number of stocks available in the dataset
Identification of highest and lowest priced stocks
Trading value analysis for each company
Volatility comparison across stocks
Profit and loss analysis using daily returns
Professional black & white financial dashboard
